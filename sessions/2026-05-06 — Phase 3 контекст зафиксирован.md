## 13 решений (compact)

**URL-pool sitemap → Camoufox pipeline:**
- **D-301** Sitemap-only URL pool (curl_cffi → slug→URLs map → Camoufox-fetch matched). Spike-validated: 1,461 slugs / 100,779 URLs plain-deliverable.
- **D-302** Полный re-crawl каждую неделю (не incremental — sitemap `<lastmod>` не отслеживает price-change).
- **D-303** Stale-SKU только в logs + `runs.stats.stale_count`. Без ops-Telegram, без `reports/stale-urls-*.txt`.

**Brand-alias coverage:**
- **D-304** Slug-эвристика от `brand_norm + aliases`, **НЕ** explicit `goldapple_slugs:` field в YAML. Trade-off: ниже manual курация, выше false-positive риск.
- **D-305** Bilingual slug-fy (ASCII + Cyrillic-preserved) + **exact** match. Никакого substring/prefix/fuzzy.
- **D-306** Skip + log в NORM-06 для viled-брендов с нулём slug-матчей.
- **D-307** Week-over-week NEW goldapple-slug diff для NORM-06 reverse-direction (не дамп всех 1,400+).

**Sanity-gate threshold M:**
- **D-308** `M = 1000` static absolute в config. ~30% спайк-оценки 3,450/week.
- **D-309** Run-to-completion + final M-gate. Без mid-run circuit-breaker.
- **D-310** Auto-suggest M в ops-Telegram после 4 недель history. **Не** auto-tune (silent drift hazard).

**Camoufox profile lifecycle:**
- **D-311** Fresh profile dir per run (tmp, snosится после). Spike-validated на холодном.
- **D-312** Integrated smoke probe ПЕРЕД crawl-фазой. Pass→crawl, fail→abort + ops-Telegram.
- **D-313** Pin exact `camoufox==135.0.1.beta24`. Manual upgrade workflow через dev-smoke + PR.

## Materially новые pattern-решения

Спайк дал stack-выбор; Phase 3 discuss добавляет operational decisions:

1. **[[Slug-эвристика для viled→goldapple, не explicit YAML]]** — пользователь предпочёл lower curation ценой false-positive, mitigated bilingual+exact-match.
2. **[[Sanity-gate M=1000 static с auto-suggest, не auto-tune]]** — manual operator-in-loop для предотвращения silent drift.
3. **[[Fresh Camoufox profile per run + integrated smoke probe]]** — defensive lifecycle, ловит fingerprint-regression до 4ч беспоsлезных fetches.
4. **[[Sitemap-only URL pool для goldapple, без brand-facet rendering]]** — Tier 0 sitemap + Tier 2 product-render asymmetric pipeline.
