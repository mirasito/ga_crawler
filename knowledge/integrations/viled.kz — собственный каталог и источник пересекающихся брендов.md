# viled.kz — собственный каталог и источник пересекающихся брендов

Местный казахстанский ритейлер — наша сторона, для команды которой строится отчёт. Меньше goldapple по объёму ассортимента.

## Двойная роль

1. **Целевой каталог** — парсим целиком (название, бренд, объём, цены, наличие, URL)
2. **Источник списка брендов** — список парсимых брендов на goldapple фильтруется по этому набору. См. [[Парсим viled целиком, goldapple только по пересекающимся брендам]]

## Anti-bot

Предположительно проще, чем у goldapple. Достаточно `curl_cffi` с `impersonate="chrome"`. Это [[Тиры anti-bot эскалации]] tier 0.

**Но проверить в Phase 1** — даже если viled "дружелюбен", парсер пишется через тот же интерфейс что и goldapple для переиспользования кода ([[Архитектура — модульный монолит на pipe-and-filter]]).

## Sitemap / feeds

Открытый вопрос: есть ли `sitemap.xml` или товарный feed. Сильно упростит URL discovery если есть.

## Brand-aliases

Топ-50 брендов viled — seed для [[Brand-alias YAML — это v1 deliverable, не v2]]. Без алиасов 30–50% пересечений теряется на стыке Cyrillic vs Latin.
