**Pre-flight checklist (operator):**
1. Confirm working directory is `C:/Users/gstorepc/projects/ga_crawler` and `uv sync` completed (Wave 0 deliverable).
2. Confirm Camoufox Firefox binary downloaded: `uv run camoufox fetch` (one-time on this dev box).
3. Confirm KZ-laptop is on Kazakhstan IP (`curl -s https://ipinfo.io/country` returns `KZ`). If not, the spike's "Tier 2 Camoufox direct, no proxy" assumption is invalid — STOP and document.

**Step 1 — Smoke probe (3 URLs, ~30s):**
```powershell
uv run python -m ga_crawler goldapple-smoke --headless false
```

Expected output: JSON with `"pass": true` and 3 entries in `diagnostics.responses`, each with `status=200`, `block=false`, `price_extracted=true`, html size > 100,000.

If `pass=false`:
- Inspect `diagnostics.responses` — note titles, sizes
- Check `camoufox_version` in diagnostics — must equal `135.0.1.beta24` (or whatever Wave 0 pinned)
- If size < 30,000 with title containing "checking" → gate-shell — Camoufox fingerprint regression. STOP. Document in checklist as `[BLOCKED-FINGERPRINT-REGRESSION]`.
- If size < 30,000 with title containing "Loading" → smoke URL went stale (de-listed). Update `config/smoke_urls.txt` (per Phase 7 ops-playbook stub) with a fresh Givenchy URL from current sitemap; rerun.
- If exception/timeout → check Camoufox install (`uv run python -c "import camoufox; print(camoufox.__version__)"`).

**Step 2 — Limited live run (~5 brands, ~50 SKUs, ~5 min at 3-5s rate-limit):**

Pick 1-2 viled brands from spike: `givenchy,jo_malone_london`. The stub orchestrator will:
1. Fetch goldapple sitemap (curl_cffi Tier 0)
2. Slug-fy and intersect — should produce some matched URLs
3. Smoke probe (re-run; takes ~30s)
4. Run loop on matched URLs (rate-limited 3-5s per fetch)
5. Apply final M-gate (use M=10 for this test — we expect <100 SKUs from limited brand list)


- `.planning/runs/42/sitemap-slugs.txt` — file with ~1,461 goldapple slugs

- stdout JSON has `"status": "success"` and `goldapple_count > 0`

**Step 3 — Verify metrics from `.planning/runs/42/runs.json`:**

- `stats["goldapple.smoke_pass"] == true`

- `stats["goldapple.fetch_failures"]` is < 10% of `fetch_count` (no sustained 429/503).
- `stats["goldapple.camoufox_version"] == "135.0.1.beta24"` (or whatever Wave 0 pinned).
- `stats["goldapple.fetch_duration_seconds"]` is reasonable (~3-5s × fetch_count for sequential).

**Step 4 — Verify NORM-06 outputs:**
