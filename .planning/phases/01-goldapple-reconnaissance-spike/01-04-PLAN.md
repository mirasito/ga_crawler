  <action>
Скачать robots.txt обоих сайтов через curl_cffi с impersonate="chrome". Использование curl_cffi (а не requests/wget) — потому что goldapple.kz может блокировать всё что не похоже на Chrome даже на robots.txt (paranoid Cloudflare config). Pause между запросами 5 секунд (минимально-консервативный per Pitfall 13).

Создать одноразовый python-script (inline через `uv run python -c` ИЛИ можно сохранить как `.planning/spikes/01-goldapple/_fetch_robots.py` если читаемее — потом удалить):

```python
import time
from pathlib import Path
from curl_cffi import requests

OUT = Path(".planning/spikes/01-goldapple/sample-payloads")
OUT.mkdir(parents=True, exist_ok=True)

targets = [
    ("viled.kz", "https://www.viled.kz/robots.txt"),
    ("goldapple.kz", "https://goldapple.kz/robots.txt"),
]

for name, url in targets:
    print(f"Fetching {url} ...")
    r = requests.get(url, impersonate="chrome", timeout=30)
    print(f"  HTTP {r.status_code}, {len(r.text)} bytes, content-type={r.headers.get('content-type', '?')}")
    out_file = OUT / f"{name.replace('.', '-')}-robots.txt"
    out_file.write_text(r.text, encoding="utf-8")
    print(f"  Saved to {out_file}")
    time.sleep(5)
```

Запустить:
```bash
uv run python -c "<inline script above>"
```
