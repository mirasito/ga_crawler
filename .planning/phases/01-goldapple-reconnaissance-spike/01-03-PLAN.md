1. Запустить smoke-test через curl_cffi с прокси, проверить что отдаётся **отличный** от локального IP (= residential поднялся):

   Сначала получить локальный (laptop) IP:
   ```bash
   uv run python -c "from curl_cffi import requests; r = requests.get('https://httpbin.org/ip', impersonate='chrome'); print('LAPTOP_IP:', r.json()['origin'])"
   ```

   Затем — IP через прокси:
   ```bash
   uv run python -c "
   import os
   from dotenv import load_dotenv
   from curl_cffi import requests
   load_dotenv('.env.local')
   proxy = f\"http://{os.environ['PROXY_USER']}:{os.environ['PROXY_PASS']}@{os.environ['PROXY_HOST']}:{os.environ['PROXY_PORT']}\"
   r = requests.get('https://httpbin.org/ip', proxies={'http': proxy, 'https': proxy}, impersonate='chrome', timeout=30)
   print('PROXY_IP:', r.json()['origin'])
   "
   ```

   Сохранить оба IP. Они **должны быть разными**. Если одинаковые — прокси не работает.

2. Дополнительно — определить geo proxy IP:
   ```bash
   uv run python -c "
   import os
   from dotenv import load_dotenv
   from curl_cffi import requests
   load_dotenv('.env.local')
   proxy = f\"http://{os.environ['PROXY_USER']}:{os.environ['PROXY_PASS']}@{os.environ['PROXY_HOST']}:{os.environ['PROXY_PORT']}\"
   r = requests.get('https://ipapi.co/json/', proxies={'http': proxy, 'https': proxy}, impersonate='chrome', timeout=30)
   ```
