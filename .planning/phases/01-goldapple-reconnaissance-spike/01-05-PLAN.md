**Шаг 1 — Sitemap.xml-first (D-11):**

Попробовать в этом порядке:
1. `https://goldapple.kz/sitemap.xml`
2. `https://goldapple.kz/sitemap_products.xml`

Через curl_cffi с impersonate="chrome", pause 5s:

```python
import time, json
from pathlib import Path
from curl_cffi import requests

OUT = Path(".planning/spikes/01-goldapple/sample-payloads")
SITEMAP_CANDIDATES = [
    "https://goldapple.kz/sitemap.xml",
    "https://goldapple.kz/sitemap_products.xml",
    # add any URLs из tos-audit.md
]

sitemap_xml = None
for url in SITEMAP_CANDIDATES:
    print(f"Trying {url} ...")
    try:
        r = requests.get(url, impersonate="chrome", timeout=30)
        print(f"  HTTP {r.status_code}, {len(r.text)} bytes")
        if r.status_code == 200 and "<urlset" in r.text or "<sitemapindex" in r.text:
            sitemap_xml = r.text
            (OUT / "goldapple-sitemap.xml").write_text(sitemap_xml, encoding="utf-8")
            print(f"  ✓ Saved {len(sitemap_xml)} bytes")
            break
    except Exception as e:
        print(f"  ✗ Error: {e}")
    time.sleep(5)

if not sitemap_xml:
    print("No sitemap accessible — fallback to pagination meta in Step 2")
```

**Шаг 2 — для каждого выбранного бренда определить product URL count:**

Стратегия 2a (если sitemap доступен):
- Для каждого бренда найти URL-pattern. Goldapple обычно использует `https://goldapple.kz/brand/<slug>` или `https://goldapple.kz/<slug>` или продуктовые URL содержат brand-slug в пути.
- Использовать `selectolax` или `xml.etree.ElementTree` для парсинга sitemap.
- Filter URLs по brand-keyword (lowercase substring match — например "estee" matches "estee-lauder", "vichy" matches "vichy"; для kyrillic брендов попробовать оба варианта).

```python
import xml.etree.ElementTree as ET

SELECTED_BRANDS = ["Estee Lauder", "Dior", "Vichy", "Bioderma", "L'Oreal"]
# нормализуем для substring search:
keywords = {b: b.lower().split()[0].replace("'", "").replace(".", "") for b in SELECTED_BRANDS}
# например: {"Estee Lauder": "estee", "Dior": "dior", "Vichy": "vichy", "Bioderma": "bioderma", "L'Oreal": "loreal"}

# Парсим sitemap
ns = {"sm": "http://www.sitemaps.org/schemas/sitemap/0.9"}
root = ET.fromstring(sitemap_xml)
all_urls = [loc.text for loc in root.iter("{http://www.sitemaps.org/schemas/sitemap/0.9}loc")]
print(f"Total URLs in sitemap: {len(all_urls)}")

per_brand = {}
for brand, kw in keywords.items():
    matched = [u for u in all_urls if kw in u.lower()]
    per_brand[brand] = {
        "url_count": len(matched),
        "source": "sitemap",
        "sample_urls": matched[:3],  # для верификации, что keyword сматчился реально
    }
    print(f"  {brand}: {len(matched)} URLs (kw={kw})")
```

Стратегия 2b (если sitemap недоступен — fallback per D-11):
- Для каждого бренда открыть страницу-листинг бренда (например `https://goldapple.kz/brand/dior` через WebFetch ИЛИ через Patchright если curl_cffi блокируется)
- Найти "Total: N товаров" / "Найдено: N" / pagination-метку (e.g., "Page 1 of 12, 24 per page" → N = 12*24 = 288)
- Записать в `per_brand[brand] = {"url_count": N, "source": "pagination", "notes": "..."}`

Если ОБА sitemap И pagination-страница блокируются Cloudflare через curl_cffi → запустить через Patchright (см. plan 01-08 для рецепта). Добавить inline в этот же script:

```python
# Fallback к Patchright если curl_cffi не справился
from patchright.sync_api import sync_playwright
with sync_playwright() as p:
    browser = p.chromium.launch(headless=True)
    context = browser.new_context()
    page = context.new_page()
    for brand, kw in keywords.items():
        slug = kw  # часто slug = lowercase brand name
        url = f"https://goldapple.kz/brand/{slug}"
        page.goto(url, wait_until="domcontentloaded", timeout=60000)
        # ищем pagination-метку
        # specific selector зависит от goldapple-разметки; в спайке OK человеку посмотреть HTML
        text = page.content()
        # save HTML sample
        Path(f".planning/spikes/01-goldapple/sample-payloads/goldapple-brand-{kw}.html").write_text(text, encoding="utf-8")
        # heuristics: ищем число рядом с "товаров" / "products" / pagination
        # ручной парсинг или пометить как "manual_count_needed"
```
